# HXWKXNN Tech Website v11.0

Site oficial da HXWKXNN Tech, com foco em privacidade, segurança, desenvolvimento e suporte técnico para Windows.

## Destaques

- Interface dark responsiva com identidade HXWKXNN Tech.
- Conteúdo completo em português e inglês.
- Serviços, planos, projetos e contato profissional.
- Página dedicada ao HXWKXNN System Manager.
- Screenshot real da versão Preview.
- Conceitos visuais de Segurança e Privacidade claramente identificados.
- Roadmap, changelog, tecnologias e área de download futuro.
- Comunidade com Instagram e Discord.
- Navegação por âncoras compensando o cabeçalho fixo.

## Tecnologias do site

- HTML5
- CSS3
- JavaScript
- SVG inline
- WebP

## Estrutura

- `home.html` — página comercial e vitrine de projetos.
- `system-manager.html` — página completa do produto.
- `sobre.html` — Felipe / HXWKXNN.
- `index.html` e `links.html` — central de links.
- `assets/screenshots/` — capturas do produto.
